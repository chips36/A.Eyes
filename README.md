# A.Eyes

__[지능형 AI 영상분석 시스템]__

 * 부족한 CCTV 관제 인력 문제 해소 & 관제 효율 개선을 위해 Computer Vision AI 를 이용한 영상관제SW
> - 칼부림 이벤트 검출
> - 폭력 (punch, kick) 이벤트 검출
> - 쓰러짐 이벤트 검출  
> - 휠체어 (사회적 약자) 검출 



# Library
